## 通过Linux控制台终端访问CLI

1. 大多数Linux发行版,都可以访问Linux虚拟控制台.通常按下ctrl + alt + f1~f7
2. 反转终端的颜色
   * `setterm -inversescreen on`
   * 也可以使用off关闭这个
   * `setterm -reset`: 默认设置
3. 两条命令
   * `setterm -background white`
   * `setterm -foreground black`
     * black, red, green, yellow, blue, magenta, cyan, white
